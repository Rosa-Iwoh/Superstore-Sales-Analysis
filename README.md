# Superstore Sales & Customer Retention Analytics

## Project Overview
This project transforms a raw transactional dataset into an interactive Executive Dashboard. It features a relational data model, advanced DAX measures, and a predictive forecasting model.

## Tools & Techniques
* **Data Engineering:** Power Query (ETL, Star Schema)
* **Data Modeling:** Power Pivot, explicit DAX Time Intelligence
* **Statistical Analysis:** Correlation Matrix, ETS Forecasting Algorithm
* **Customer Analytics:** Custom RFM (Recency, Frequency, Monetary) Segmentation

## Key Business Insight: The Discount Trap
Statistical analysis revealed that deep discounts negatively correlate with profit margins (-0.219) while failing to drive higher order volumes (0.008). **Recommendation:** Cap discounts at 20% for low-margin categories.
